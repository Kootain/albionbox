# Guild（服务器）

官方接口列表：https://developer.kookapp.cn/doc/http/guild

## 常用能力

- 获取当前用户/机器人加入的服务器列表
- 查看服务器详情
- 管理成员（踢出、改昵称等）

## 典型接口（按业务常用度）

- `GET /api/v3/guild/list`
- `GET /api/v3/guild/view`
- `GET /api/v3/guild/user-list`
- `POST /api/v3/guild/nickname`
- `POST /api/v3/guild/kickout`
- `POST /api/v3/guild/leave`
