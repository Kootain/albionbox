# Channel（频道）

官方接口列表：https://developer.kookapp.cn/doc/http/channel

## 常用能力

- 列出服务器下的频道
- 查看频道详情
- 创建/删除频道
- 发送频道消息（与 Message 资源页同能力）

## 典型接口

- `GET /api/v3/channel/list`
- `GET /api/v3/channel/view`
- `POST /api/v3/channel/create`
- `POST /api/v3/channel/delete`
- `POST /api/v3/channel/move-user`（语音相关）
